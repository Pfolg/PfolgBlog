# 简介

此程序是我的一个练习作品，单纯是为了提升编程水平，次要是为了做一个NirCmd的Gui，其实主要成分还是Gui，核心代码就两三行。

主要是Gui，功能基于 [nircmd v2.87] 实现，程序本身不提供一些重要的功能。

# 关于NirCmd

http://nircmd.nirsoft.net/

# 关于此程序

看源码就行了，基于Python，使用tkinter制作Gui。

我只是选择性的对一些命令进行了Gui化，如果你还想要完全释放NirCmd的能力，你可以阅读官方文档进一步学习。
