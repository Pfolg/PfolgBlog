# 下载

+ 克隆本仓库到本地（不推荐）；

+ 若要下载单个文件夹，复制该文件夹的网址，粘贴入[DownGit](https://downgit.github.io/#/home)中，选择download即可。
+ 或者选择特定的文章再在页面上利用GitHub的下载功能。
